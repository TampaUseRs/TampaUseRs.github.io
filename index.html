Discussion Board

We have a discussion board available on GitHub for #rstats related questions, posts, discussions and announcements.

To join, request access to GitHub TampaUseRs organization by sending your GitHub username to @UseRTampa on Twitter or the TampaUseRs organizers via email or Meetup. The discussion board is under Teams > Everyone > Discussion.

Speaker Resources
